Software Engineer Trainee

Location: Coimbatore, Tamil Nadu

Department: Software Development

Type: Full-Time / On-site

About the Role

Join our development team as a Software Engineer Trainee to build scalable applications and contribute to real-world software projects under professional guidance.

Responsibilities

  • Assist in designing, coding, and testing software applications.
  • Collaborate with senior developers to implement new features.
  • Participate in code reviews and contribute to best coding practices.
  • Troubleshoot bugs and maintain existing applications.

Skills Required

  • Strong understanding of programming languages such as Python, Java, or C++.
  • Basic knowledge of web technologies (HTML, CSS, JavaScript).
  • Familiarity with version control systems (Git/GitHub).
  • Problem-solving mindset and eagerness to learn new technologies.

Eligibility

  • B.E/B.Tech (CSE, AI & DS, IT, ECE) or related fields.
  • 0–1 year of experience or final-year students are welcome.
  • Good communication and teamwork skills.

Perks

  • Mentorship from senior engineers.
  • Flexible working environment and learning opportunities.
  • Access to internal workshops and certification programs.
